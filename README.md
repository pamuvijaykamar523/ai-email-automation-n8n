# AI Email Automation n8n

This project automates lead collection and email workflows using n8n.

## Features
- Lead collection
- Automated email sending
- CRM integration
- Workflow automation

## Screenshots
Project screenshots are available in this repository.

## Tools Used
- n8n
- Gmail
- Webhooks
- CRM
## Workflow

1. User submits lead form
2. Webhook receives data
3. Lead is processed
4. CRM is updated
5. Automated email is sent
